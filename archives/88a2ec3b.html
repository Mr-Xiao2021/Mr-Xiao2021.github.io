传统的queue

在C++中，queue 是标准模板库（STL）中的一个容器适配器，它提供了先进先出（FIFO）的队列操作。queue<int> 表示一个存储整数的队列。以下是 queue 的一些基本用法：

  1. 包含头文件：
    要使用 queue，首先需要包含相应的头文件。

  2. 声明队列：
    声明一个 queue 的实例。

  3. 添加元素：
    使用 push() 方法向队列的末尾添加元素。

  4. 访问队首元素：
    使用 front() 方法可以访问队列的第一个元素，但不能修改它。

  5. 访问队尾元素：
    使用 back() 方法可以访问队列的最后一个元素，但不能修改它。

  6. 移除元素：
    使用 pop() 方法从队列的前端移除元素。注意，pop() 不返回任何值，如果你想在移除前访问元素，应该先用 front()。

  7. 检查队列是否为空：
    使用 empty() 方法检查队列是否为空。

  8. 获取队列的大小：
    使用 size() 方法获取队列中的元素数量。

  9. 交换两个队列的内容：
    使用 swap() 方法可以交换两个队列的内容。
